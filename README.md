# iot_fridgedb

A new Flutter project.

#### 这是一个用于连接aws database的简易app，

是一个IoT课程的期末专案的一部分，冰箱DB，通过image AI识别冰箱中的物品并上传db，然后app可以直接读取db的资料，

便于查看当前冰箱有什么东西，什么时候拿走了什么东西。

当然前面的全部都是假想，实际上识别冰箱内的食品之类的东西就三周时间断断续续的根本搞不完
